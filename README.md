# Naša Hrvatska — Our Croatia 🇭🇷

Croatian language and culture learning app for families moving to Croatia.

**Live:** [nasahrvatska.com](https://nasahrvatska.com)

## Quick Start

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
# Deploy dist/ folder to Netlify
```

## Refactoring

See `CLAUDE-CODE-INSTRUCTIONS.md` for the full refactoring roadmap.

## Tech Stack

- React 18 (Vite)
- Firebase Auth + Firestore
- Azure Neural TTS (via Netlify Function)
- Netlify hosting
