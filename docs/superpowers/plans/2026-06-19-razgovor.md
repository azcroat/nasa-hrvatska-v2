# Phase 7a — Razgovor (Talk / AI) Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (or subagent-driven-development) to implement task-by-task. Steps use checkbox (`- [ ]`).

**Goal:** Re-present the AI tab as **Razgovor** — five family conversation partners + a "Danas" recommended chat + an "Alati" utilities shelf — without orphaning any of the 10 existing AI entry points and without changing any conversation/tool screen's behaviour.

**Architecture:** A re-presentation layer in `src/components/razgovor/`. A `partners.ts` registry maps every AI entry point to a partner mode or the Alati shelf and computes `recommendedChat()`. `RazgovorTab` (Today + partners + Alati) + `PartnerScreen` (interior) replace `AITab`. Partner "Počni razgovor" preselects a persona (`localStorage maja_persona`) and opens the existing `maja` conversation. Existing screens reused unchanged. A no-orphan unit test proves coverage.

**Tech Stack:** React 18 + Vite + TS; plain CSS + tokens (no Tailwind); Outfit + Playfair; Vitest + @testing-library/react; Playwright e2e.
